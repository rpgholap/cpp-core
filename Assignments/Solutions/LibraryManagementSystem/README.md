# Library Management System

A C++ Library Management System implementing advanced concepts such as object-oriented programming, inheritance, polymorphism, templates, standard template library (STL), exception handling, and file persistence.

## Features
1. Book Management - Add, remove, search, and display books
2. Member Management - Register, remove, search members
3. Issue/Return Books - Track book transactions
4. Fine Calculation - Calculate fine for late returns
5. Data Persistence - Save and load data from files

## Folder Structure
- `headers/` - Contains all class declarations (`*.h`)
- `src/` - Contains all definitions and main logic (`*.cpp`)
- `data/` - Contains all stored data in plain text
